# Governance

This directory records local project governance for the cognitive-disciplines
plugin effort. GitHub issues and the GitHub Project board are the external
tracking surface; these files are the repo-local record of decisions,
intentions, open questions, and tasks.

## Ledgers

- `decisions/`: accepted project decisions and their rationale.
- `intentions/`: active intentions that guide near-term action.
- `open_questions/`: unresolved questions that must be answered before or during implementation.
- `tasks/`: local task records that may map to GitHub issues or PRs.
- `operating-norms.md`: working practices for the governed plugin effort.

## Rule

Major plugin work should point back to a decision, intention, task, or open
question here, and to a GitHub issue or PR when externally visible tracking is
needed.
