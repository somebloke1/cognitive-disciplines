# Mission Primer

## Mission

Develop the cognitive cycle into a full-fledged Codex plugin.

The plugin must transform the current P1-P4 discipline from a set of skills and
planning artifacts into a harnessed, functional Codex plugin with durable
governance, GitHub tracking, curriculum-primed specialists, artifact-first
handoffs, acceptance gates, recursion contracts, and verification.

## Order Of Work

1. Establish and preserve governance practices and norms.
2. Use GitHub issues, PRs, and Project #7 as the external project surface.
3. Resolve plugin format and installation target from current evidence.
4. Scaffold a valid Codex plugin.
5. Package skills, curriculum primers, templates, references, and harness
   resources.
6. Validate the plugin structurally and behaviorally.
7. Open PRs for reviewable milestones.

Do not skip from existing skills directly to plugin claims. The harness must be
made explicit.

## Required Capabilities

The final plugin should provide:

- P1 Attend.
- P2 Inquire.
- P3 Judge.
- P4 Decide.
- P1 Data Curator for multi-agent P1 sets.
- P2 Possibility Integrator for multi-agent P2 sets.
- P3 Dialectician for multi-agent P3 sets.
- P4 Ethics Sage for multi-agent P4 sets.
- Full cognitive-cycle orchestration.
- GitHub Project coordination support.
- GraphQL efficiency support for ProjectV2 work.

## Behavioral Standard

Prompt labels are insufficient. The plugin must reinforce behavior through:

- staged role priming;
- grounding artifacts;
- packet templates;
- ordered archives;
- acceptance and rejection gates;
- recursion ledgers;
- controller transition rules;
- final verification against current evidence.

## Core Distinction

Preserve the distinction between deterministic harness mechanics and
language-model cognitive judgment.

Harness mechanics include manifests, phase order, packet shapes, archives,
acceptance gates, recursion budgets, routing, project tracking, and validation.

Cognitive judgment includes evidence relevance, possibility distinctness,
sufficiency judgment, recursion choice, responsible decision, and residual-risk
assessment.
