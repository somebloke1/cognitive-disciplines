# Intention 0001: Develop Practices And Norms First

- `status`: active
- `date`: 2026-06-21
- `github`: https://github.com/somebloke1/cognitive-disciplines/issues/4
- `decision`: governance/decisions/0001-operate-as-governed-plugin-controller.md

## Intention

Develop the practices and norms surrounding the plugin-development requirements
before transforming the cognitive discipline into a harnessed, fully-developed
Codex plugin.

## Immediate Actions

- Establish local governance ledgers.
- Define operating norms and acceptance gates for plugin work.
- Track the work through GitHub issues and the GitHub Project board.
- Commit the governance baseline.
- Only then begin plugin transformation.

## Completion Signal

This intention is satisfied when the repository has committed governance
artifacts that define decisions, intentions, tasks, open questions, operating
norms, and the transition conditions for beginning plugin implementation.
