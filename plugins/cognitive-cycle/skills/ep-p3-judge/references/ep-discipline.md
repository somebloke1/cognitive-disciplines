# EP Discipline Reference

Core pattern: `R(P1 -> P2 -> P3 -> P4 -> R)`.

- P1: Be attentive. Characterize what is given with high fidelity.
- P2: Be intelligent. Generate multiple grounded possibilities.
- P3: Be reasonable. Judge what is true and whether recursion is required.
- P4: Be responsible. Declare and bring into being within authority.

EP audit lens:

- A scheme of recurrence is a recurring operational pattern that sustains itself while conditions hold.
- A conditioned series is an ordering where later schemes require earlier schemes to function.
- Emergent probability combines probabilities of emergence and survival across conditioned schemes.
- Audit for functioning schemes, possible schemes, survival risk, emergence probability, breakdown cascades, blind alleys, defenses, floating-population components, scale/time strategy, and development trajectory.

Development lens:

- An Integrator maintains a stable scheme.
- An Operator modifies the underlying manifold so a higher integration can emerge.
- Genuineness requires admitting the tension between limitation and transcendence; scotosis hides that tension.
