# Same-Phase Agent Differentiation

Use this reference when a team or legion-scale cycle assigns more than one
agent to the same P1, P2, P3, or P4 level. Same-phase agents should not be
interchangeable copies. The controller should assign differentiated focal
emphases, then route the peer packets through the matching same-phase
integration skill before the next phase.

The harness can require differentiated assignments and packet fields. The
substance of the differentiation remains cognitive judgment: whether an
attentional focus is relevant, a creative line is authentic, a judgment is
warranted, or a value emphasis is responsible cannot be decided by pattern
matching.

## Controller Assignment Rule

For each same-phase agent set, record:

- `agent_id`
- `phase`
- `focal_emphasis`
- `source_or_packet_scope`
- `tools_or_methods`, if materially different
- `forbidden_overlap`: what the agent should avoid duplicating
- `expected_packet`

The focal emphases should create useful cognitive contrast without fragmenting
the shared orienting question.

## P1 Differentiation: Attentional Coverage

Differentiate P1 agents by attentional access to the given:

- source strata: code, docs, logs, tests, issues, conversations, artifacts;
- tool pathway: static inspection, command output, trace review, UI readback,
  archive review, external source check;
- evidential question: presence, absence, contradiction, provenance,
  breakdown, dependency, history, or boundary condition.

P1 agents should still share the same P1 discipline: observe before explaining,
separate evidence from inference, and mark negative search space. The curator
consolidates without inventing missing data.

## P2 Differentiation: Formulative Impetus

Differentiate P2 agents by creative and formulative impetus:

- architecture-forming possibilities;
- process, governance, or sequencing possibilities;
- failure-mode, risk, or reversal possibilities;
- developmental or emergence-path possibilities;
- conservative, disruptive, or boundary-expanding formulations.

P2 agents should produce authentic lines of possibility grounded in P1, not
renamed variants. The P2 integrator orders the field for P3 while preserving
genuine creativity, especially lines that are unusual, uncomfortable, or not
yet dominant.

## P3 Differentiation: Adversarial Judgment

Differentiate P3 agents by judgment posture:

- affirmative sufficiency case;
- skeptical insufficiency case;
- contradiction and burden-of-proof review;
- comparative warrant ranking;
- recursion-first review.

P3 agents should be at least implicitly adversarial: each should test whether
the substrate can bear the proposed judgment rather than merely endorse a
plausible answer. The dialectician reconciles competing judgments by comparing
warrants, substrate fidelity, uncertainty handling, and recursion discipline.

## P4 Differentiation: Value Emphasis

Differentiate P4 agents by responsible value emphasis within the manifest mode:

- prudence and reversibility;
- justice, fairness, and affected parties;
- effectiveness and execution force;
- stewardship of future optionality;
- duty to stated constraints, authority, and user intent.

P4 agents should not invent outside objectives or exceed authority. They should
make the value emphasis explicit so the sage can evaluate the values, compare
how each decision weights consequences and obligations, and determine the best
authorized course.

## Integration Obligations

Same-phase integrators must preserve the differentiated contribution before
consolidating:

- P1 curator: preserve source/tool coverage, absences, contradictions, and
  minority observations.
- P2 integrator: preserve authentic creative lines; remove only duplicates,
  pseudo-differences, and unsupported drift.
- P3 dialectician: preserve competing judgments long enough to compare their
  warrants; do not compromise away real disagreement.
- P4 ethics sage: evaluate value emphases as value emphases; do not treat
  consensus, forcefulness, or convenience as ethical adequacy.

An integration packet should name the peer focal emphases and explain how they
were preserved, merged, bracketed, or rejected.
